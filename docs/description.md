SubGuard

A banking-style app that gives each subscription its own virtual card with a spending cap, renewal alert, and one-click freeze. It solves the annoying problem of surprise charges, free trials turning paid, and subscriptions that are hard to cancel.

Why it stands out:

Real pain point
Feels useful immediately
Strong security story
Clean dashboard-heavy frontend

Core features:

Sign up / login
Create virtual cards for merchants
Set per-card spending limits
Freeze or delete a card instantly
View upcoming renewals and past charges
Get alerts for unusual or over-limit charges

Security features:

JWT auth
BCrypt password hashing
Owner-only access
OTP for card freeze/delete or limit changes
Audit log for sensitive actions
Failed login lockout

Frontend value:

Merchant cards grid
Spending and renewal dashboard
Alerts center
Transaction timeline
Security settings page

Portfolio line:
A secure subscription control app built with React and Spring Boot that lets users create merchant-specific virtual cards, cap recurring charges, and instantly block unwanted payments.
